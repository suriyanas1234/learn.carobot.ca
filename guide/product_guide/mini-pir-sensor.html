How to the Use the Mini PIR Sensor Module (HC-SR505)

The mini PIR sensor module is a sensor that can detect motion through the use of infrared light. In this guide, we will be using the Mini PIR Sensor Module (HC-SR505) that is found in our shop.

Step 2: Start the serial monitor and activate the pin that is connected to the signal output

void setup(){
  Serial.begin(9600);
  pinMode(signalPin, INPUT);
}

Step 3: Write down some code that will print on the serial monitor whether the sensor senses movement or not

void loop(){
  if(digitalRead(signalPin)==HIGH){
    Serial.println("Movement detected");
  }else{
    Serial.println("Did not detect movement");
  }
  delay(1000);
}

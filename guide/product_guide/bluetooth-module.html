HC-05 Bluetooth Module Guide

The HC-05 Bluetooth module (and related modules) is a very popular module for wireless communication due to its price and capabilities.

In this article, we‚Äôll talk about how to wire and setup the module for operation with an Arduino board.

Wiring

Most HC-05 modules require a 5V operating voltage. However, the TX/RX pins work at 3.3V.

Although most modules have on-board current limiting circuit which allows the TX/RX pins to work with 5V signal, a 5V to 3.3V circuit (a simple voltage divider) at the Arduino TX pin is recommended.

There are a few version of the HC-05 module in the market but for this article, we‚Äôll work with HC-05 Bluetooth module, the one we carry in our shop.

Programming‚Äã

The default setting of the HC-05 is suitable for most cases. However, if you want to change some of the setting such as increase the baud rate for higher speed data, you‚Äôll need to enter HC-05‚Äôs command mode. Some HC-05 modules have a command pin built onto the board, the one we carry in our shop use a button to enter command mode instead.

To enter command mode, power off the HC-05 module, hold the command mode button (or pull pin 34 high if the HC-05 module doesn‚Äôt have a AT button), then power the HC-05 module back on. Once you are in command mode, the module‚Äôs LED led will blink at about 2 seconds interval.

Pull pin 34 high (all 1k resistor).

Now that your HC-05 is in command mode (also known as AT mode), you can use the Arduino sketch below to program your Bluetooth module.

#include <SoftwareSerial.h>

const int txPin = 9;
const int rxPin = 10;

SoftwareSerial BTSerial(rxPin, txPin); // RX, TX

void setup() {
  Serial.begin(9600);
  Serial.println("Enter AT commands:");
  BTSerial.begin(38400);
}

void loop() {
  if (BTSerial.available())
    Serial.write(BTSerial.read());
  if (Serial.available())
    BTSerial.write(Serial.read());
}

HC-05‚Äôs AT mode baud rate is fixed at 38400, however, the BT communication baud rate can be changed from the default 9600 to as fast as 115200 (or faster, but we never tested).

Here is a document explaining the HC-05‚Äôs AT command.

Example‚Äã

  1. Once you uploaded the sketch onto your Arduino, open the serial monitor at 9600 baud rate, select CR+NL, and you should be able to send AT commands to the HC-05.
  2. Every time you transmit ‚ÄúAT‚Äù (without the quotes), the HC-05 should reply with ‚ÄúOK‚Äù
  3. To check your HC-05‚Äôs version, enter ‚ÄúAT+VERSION‚Äù
  4. To change the name of the module, enter ‚ÄúAT+NAME=MYBTNAME‚Äù
  5. To check the current password, enter ‚ÄúAT+PSWD‚Äù
  6. To change the password, enter ‚ÄúAT+PSWD=2345‚Äù
  7. To check the current baud rate, enter ‚ÄúAT+UART‚Äù
  8. To change the BT baud rate to 115200, 1 stop bit, 0 parity, enter ‚ÄúAT+UART=115200,1,0‚Äù
